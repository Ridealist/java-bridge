## 다리건너기

### 기능 목록

1. 다리 길이 입력
    1. 예외 처리
        - [x] `3이상` `20이하` 가 아니면 예외 발생'
        - [x] `숫자`가 아니면 예외 발생

2. 다리 생성
    - [x] `입력된 숫자`만큼의 길이 생성

3. 생성된 다리의 여부 설정
    - [x] `1`과 `0`의 여부를 각각 설정한다

4. 이동할 위치 입력
    1. 예외 처리
        - [x] `U` 또는 `D`가 아닌 경우
        - [x] `공백`인 경우

5. 게임 결과 출력
    1. 실패할 경우 다시 `1.`부터 실행

6. 시도 횟수 계산
